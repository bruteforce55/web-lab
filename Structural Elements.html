HTTP/3

HTTP/3 or simply H3 is the upcoming third major version of the Hypertext Transfer Protocol used to exchange information on the World Wide Web, succeeding HTTP/2. HTTP/3 is a draft based on a previous RFC draft, then named "HTTP over QUIC". HTTP/3 is the new standard for the web, enabling faster, more reliable, and more secure connections to web endpoints like websites and APIs.

What is HTTP?

HTTP means Hypertext Transfer Protocol. HTTP is the underlying protocol used by the World Wide Web and this protocol defines how messages are formatted and transmitted, and what actions Web servers and browsers should take in response to various commands.
